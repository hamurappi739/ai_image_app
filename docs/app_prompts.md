# Промпты приложения (источник текстов)

**Канонический файл** с расширенными текстами для генерации. Используется при разработке и при обновлении строк в приложении.

**Статус:** тексты **встроены в приложение** (централизованный модуль `frontend/lib/data/app_prompts.dart`).

| Раздел UI | Короткий текст | Длинный текст | Куда попадает в приложении |
|-----------|----------------|---------------|----------------------------|
| **Фото по шаблону** (17 шаблонов) | Описание на карточке | Большой промпт | **«Выбрать»** → поле **«Свой запрос»** (`requestDescription`) |
| **Фотосессии** (15 стилей) | Описание на карточке | Style prompt (серия из 3 фото) | `POST /photoshoots/generate` — поле **`description`** |
| **Своя фотосессия** (5 чипов) | Подпись чипа | Полный текст | Поле описания в модалке при нажатии на чип |

**Backend:** для готовых фотосессий длинный текст уходит в **`description`**; системные правила качества — в `gemini_quality_instructions.py`. Fallback-инструкции стилей — в `photoshoot_styles.py` (если `description` не передан).

**Как обновлять:** править тексты здесь → синхронизировать `frontend/lib/data/app_prompts.dart` (и при необходимости `photoshoot_styles.py` для новых `style_id`).

Дубликат для Cursor-задач: [app_prompts_for_cursor.md](app_prompts_for_cursor.md).

3. Своя фотосессия — чипы

Эти тексты можно подставлять в поле описания, когда пользователь нажимает быстрый чип в модалке “Своя фотосессия”.

4. Короткая техническая заметка для вставки

• Для карточек “Фото по шаблону” лучше подставлять большой промпт целиком в поле “Свой запрос”.

• Для готовых “Фотосессий” эти тексты лучше использовать как style prompt: пользователь выбирает стиль, добавляет фото, а генерация делает 3 варианта в одном направлении.

• Для чипов “Своей фотосессии” тексты можно держать короче, но обязательно оставлять блок про сохранение лица и отсутствие искажений.

• Если в приложении появится отдельный системный negative prompt, повторяющийся блок “без искажений лица, рук, пальцев...” можно вынести туда, чтобы сами промпты стали компактнее.
