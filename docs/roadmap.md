# Roadmap

1. **Backend MVP** — базовый API, health-check, структура приложения
2. **Gemini integration** — генерация изображений через Gemini API
3. **Supabase credits** — учёт пользователей и баланса кредитов
4. **Flutter UI** — мобильный клиент, экраны генерации и истории
5. **RuStore Billing** — оплата и пополнение кредитов
6. **Release** — публикация в RuStore, мониторинг, поддержка
