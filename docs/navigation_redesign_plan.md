# Navigation Redesign Plan

План и фактический статус UX-redesign навигации под аудиторию **женщин 40+**.

---

## Цель

- Упростить путь пользователя: **от простого к сложному**
- Убрать нижние вкладки; основная навигация — **burger/drawer** слева сверху
- Заменить технические названия на понятные русские слова
- Показывать **баланс** в меню и шапке — без слов credits / package

---

## Пользовательский путь (актуальный)

### Фото по шаблону

**Главная** → **«Начать создавать»** → **Фото по шаблону** → выбор шаблона → **Свой запрос** (описание подставлено) → фото → **«Создать фото»** → **Готовые фото**

### Фотосессии

**Фотосессии** → выбор стиля → фото → **«Создать фотосессию»** → **Готовые фото**

### Покупка при необходимости

**Купить** — когда закончились фото или фотосессии; мягкие подсказки и диалоги ведут в этот раздел.

### От простого к сложному (общий принцип)

1. **Фото по шаблону** — самый простой вход
2. **Фотосессии** — серия из 3 фото в стиле
3. **Свой запрос** — своя идея с нуля

## Текущий drawer menu (burger)

| Пункт | Статус | Примечание |
|-------|--------|------------|
| **Приветствие** | ✅ | Имя / «Здравствуйте», email |
| **Ваш баланс** | ✅ | Фото, Фотосессии, Бесплатные (если есть); кнопка **«Купить»** |
| **Главная** | ✅ | Welcome-экран; **«Начать создавать»** → **Фото по шаблону** |
| **Фото по шаблону** | ✅ | Шаблоны по категориям; **«Выбрать»** → **Свой запрос** |
| **Фотосессии** | ✅ | Подборки стилей; **«Своя фотосессия»** сверху |
| **Трендовые фотосессии** | ✅ | Прокрутка к блоку трендов на экране **Фотосессии** (отдельного экрана нет) |
| **Свой запрос** | ✅ | Фото → описание → **«Создать фото»** |
| **Готовые фото** | ✅ | Результаты, success-блок после генерации, быстрые действия |
| **Купить** | ✅ | 1 фото = 10 ₽, 1 фотосессия = 100 ₽, фотосессия = 3 фото |
| **Профиль** | ✅ | Личный кабинет: вход, баланс |
| **Помощь** | ✅ | Hub с темами по разделам |

**Шапка экранов:** burger слева; справа — компактный индикатор **«Фото: N»** (на узком экране может скрываться; полный баланс всегда в drawer).
