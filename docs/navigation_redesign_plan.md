# Navigation Redesign Plan

План и фактический статус UX-redesign навигации под аудиторию **женщин 40+**.

---

## Цель

- Упростить путь пользователя: **от простого к сложному**
- Убрать нижние вкладки; основная навигация — **burger/drawer** слева сверху
- Заменить технические названия на понятные русские слова

---

## Пользовательский путь (актуальный)

1. **Фото по шаблону** — самый простой вход (готовые карточки, описание подставляется автоматически)
2. **Фотосессии** — серия из 3 фото в выбранном стиле
3. **Свой запрос** — для своей идеи: добавить фото → написать описание → создать фото

---

## Структура меню (drawer)

| Пункт меню | Статус | Примечание |
|------------|--------|------------|
| **Главная** | ✅ | Welcome-экран; кнопка «Начать создавать» → **Фото по шаблону** |
| **Фото по шаблону** | ✅ | 6 шаблонов; «Выбрать» → **Свой запрос** с готовым описанием |
| **Фотосессии** | ✅ | Популярные / другие стили; блок «Создать свой образ» сверху |
| **Свой запрос** | ✅ | Упрощённый flow: фото → описание → «Создать фото» |
| **Готовые фото** | ✅ | Бывш. «Галерея»; empty state с кнопками в шаблоны и фотосессии |
| **Купить** | ✅ | Бывш. «Пакеты»; баланс, наборы, «Своя сумма» |
| **Профиль** | ✅ | Вход / регистрация / баланс |
| **Помощь** | ✅ | Hub с темами по каждому разделу |

### Alias: «Трендовые фотосессии»

Если пункт **«Трендовые фотосессии»** есть в drawer — он ведёт на экран **Фотосессии** (тот же `AppSection.photoshoots`). Отдельного экрана нет; пользователь видит блок **«Популярные фотосессии»** вверху списка.
